# 🚀 JavaScript Learning Journey

**Welcome!**  
This repository is a structured collection of notes, examples, and practice code to help you learn JavaScript from basics to advanced concepts.

---

## 📚 What You’ll Discover Here
Step-by-step organized, so you can learn core JavaScript concepts systematically.

---

<details>
<summary>💡 Basics</summary>

Learn the building blocks of JavaScript:  

- 📝 **Variables**: `var`, `let`, `const`  
- 🔢 **Data Types**: String, Number, Boolean, Null, Undefined, Symbol  
- ⚡ **Operators**: Arithmetic, Assignment, Comparison, Logical, Ternary  
- 🔄 **Type Conversion**: Implicit & Explicit conversions  

</details>

<details>
<summary>🔤 Strings & Numbers</summary>

Work with text and numbers effectively:  

- 📝 **String Methods**  
- 🏷 **Template Literals**  
- 🔢 **Number Methods**  
- 📏 **Math Object**  

</details>

<details>
<summary>🔍 Control Flow</summary>

Control the execution of your code:  

- ❓ **Conditional Statements**: `if`, `else if`, `else`, `switch`  
- 🔁 **Loops**: `for`, `while`, `do...while`  
- ⛔ **Loop Control**: `break`, `continue`  

</details>

<details>
<summary>🛠 Functions</summary>

Master reusable code blocks:  

- 🖊 **Function Declarations & Expressions**  
- ⚡ **Arrow Functions**  
- 📥 **Parameters & Return Values**  
- 🔒 **Scope & Closures**  

</details>

<details>
<summary>🗂 Arrays & Objects</summary>

Handle collections and structured data:  

- 📋 **Array Methods**: `map`, `filter`, `reduce`, `every`, `forEach`, etc.  
- 🏷 **Object Properties & Methods**  
- ✂️ **Destructuring**  
- 📦 **Spread Operator**  

</details>

<details>
<summary>🌐 DOM Manipulation</summary>

Interact with the browser and HTML:  

- 🔍 **Selecting Elements**  
- 🖱 **Event Listeners**  
- 🎨 **Changing Styles & Content**  
- 📝 **Form Handling**  

</details>

<details>
<summary>🚀 Advanced Topics</summary>

Level up your JavaScript skills:  

- ✨ **ES6+ Features**  
- ⏳ **Promises**  
- 🌀 **Async/Await**  
- 🌐 **Fetch API**  
- 📦 **Modules**  
- ⚠️ **Error Handling**  

</details>
